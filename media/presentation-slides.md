## Slide 13: What are Metaskills?

**Logo**: `academy-logo.png` (top-right)
**Background**: dark (#0a0a0a)

**Content:**
```
Metaskills
━━━━━━━━━━

Skills that help you THINK:
  /interview      → 95% confidence about what you actually want
  /problem-statement → Frame the problem before solving it
  /compare-options   → Structured pro/con with a recommendation

Skills that BUILD skills:
  /skill-creator  → Turns your workflow into a reusable skill
  /skill-audit    → Reviews and improves existing skills

Meta: skills that create skills that create value
```

**Speaker notes:** "This is the final level. Skills that help you think clearly, and skills that build NEW skills. The skill-creator will interview you about your most repetitive workflow, frame the problem, design the solution, and generate a working SKILL.md."
